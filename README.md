# DC ELEMENTOR ADD-ONS
Wordpress elementor widget plugin
A widget collection for Elementor page builder plugin.

## Add-ons
* Loop post
* Icon text
* Temoin
* Gallery Stick (deprecated)
* Slider Temoin
* Bulle
* Bloc picto text
* Bloc carre
* Scrollmagic
* Slider Post
* Slider Gallery
